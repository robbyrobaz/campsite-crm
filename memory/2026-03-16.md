## Kanban Spawn Issues Fixed (05:46-05:56 MST)

**Problem:** All builder spawns failing with `ENOENT` error.

**Root causes (3 issues fixed):**
1. **PATH:** Node spawn() doesn't inherit shell PATH — used full binary paths
2. **Symlinks:** `/home/rob/.local/bin/claude` symlink not followed — used real path `/home/rob/.local/share/claude/versions/2.1.76`
3. **Relative cwd:** `project_path: "blofin-moonshot-v2"` failed when kanban cwd was `kanban-dashboard` — resolve to absolute path before spawn

**Fixes committed:**
- `kanban-dashboard/server/index.ts` — full paths, absolute cwd resolution
- All 3 commits pushed to `robbyrobaz/claw-kanban`

**Lesson:** Node spawn() cwd option requires absolute paths when binary paths are also absolute. Relative paths cause ENOENT even with correct PATH.
