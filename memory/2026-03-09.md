- **005 Large Print Momentum**: average bar size — should be individual whale ticks
- **006 Tape Streak**: bar-grouped ticks — should walk full tick stream across bar boundaries
- **008 Stacked Book**: NBBO bid/ask aggregated to bars — should use nq_depth 5-level DOM
- **001/003/009/010/011**: bar-level delta/CVD acceptable per spec
