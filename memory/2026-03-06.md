## ORB Engine — Bugs Found & Fixed (Morning Session)

### Bug 1: Bar timing (CRITICAL)
- Engine used `bar_minute = now_utc.replace(second=0)` for data lookups
- This looks up the CURRENT (still open) bar — always returns None
- Fix: decouple `current_minute` (timing checks) from `bar_minute = current_minute - 1min` (data lookups)
- Result: engine now reads the last COMPLETED bar correctly

### Bug 2: Trail fires on entry bar (CRITICAL)
- After entering, trail check ran on the same bar used for entry decision
- That bar's H/L includes pre-entry price action (e.g. big wick before our fill)
- Fix: skip trail check when `bar_minute <= entry_bar` — start trail on NEXT bar
- `entry_bar` now stored in state JSON
