### Key Technical Lesson
**Blofin API pagination** (OKX-style):
- `after=<ts_ms>`: returns candles OLDER than timestamp (backward pagination)
- `before=<ts_ms>`: returns candles NEWER than timestamp (forward, mostly broken for pagination)
- `limit`: max 300 per request
- CCXT maps `until` → `after`, but `since` is only client-side filtering (useless)
- Data returned in DESCENDING order (newest first)
